💰 FinanceAgent Azure 연동 구조 · finance_agent.py
Azure OpenAI · Chat Completion gpt-4o 계열 · 3단계 파이프라인 전체 담당
파라미터 추출 → 시뮬레이션 해석 → 응답 생성
FinanceSimulationPlugin — Azure 미사용 (Pure Python): 몬테카를로 시뮬레이션 10,000회는 순수 Python 수치 연산으로 처리됩니다. NumPy / 표준 라이브러리만 사용하며 외부 네트워크 호출이 없습니다. monte_carlo_simulation(), investment_recovery() 모두 로컬 실행.
3단계 LLM 파이프라인: ① 자연어 질문 → Azure OpenAI Chat으로 파라미터 JSON 추출 → ② FinanceSimulationPlugin이 수치 시뮬레이션 로컬 실행 → ③ 시뮬레이션 결과를 다시 Azure OpenAI Chat에 전달하여 한국어 해석 생성. VariableExtractor가 응답 후처리로 재무 변수를 세션에 누적하여 multi-turn 대화 지원.
