⚠️ 핵심 데이터 변환 포인트 — V_LAW_TO_ADM (Oracle VIEW)
VWorld WFS는 법정동 코드(emd_cd)를 반환하지만, Oracle의 매출 테이블 SANGKWON_SALES는 행정동 코드(adm_cd) 기준으로 집계됩니다. 법정동과 행정동은 1:N 관계이므로, DongMappingDAO가 시작 시 V_LAW_TO_ADM 뷰를 로드하여 Confidence Score로 최적 매핑을 선택한 뒤 WFS GeoJSON 피처에 adm_cd를 주입합니다. 이 변환 없이는 지도 클릭 → 매출 데이터 연결이 불가합니다.
