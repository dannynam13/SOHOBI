===================================================
MongoDB 설치 & 환경설정 정리
===================================================

-----------------------------------------------------
1. MongoDB 설치 [서버에서]
-----------------------------------------------------
- https://www.mongodb.com/try/download/community
- Community Server → Windows 선택 → Download
- setup.exe 실행 → Complete → Next Next 설치

-----------------------------------------------------
2. DB 파일 저장 폴더 확보 [서버에서]
-----------------------------------------------------
- 탐색기에서 폴더 직접 생성
- 예) C:\Sohobi\mdb

-----------------------------------------------------
3. MongoDB 기동 [서버에서]
-----------------------------------------------------
- MongoDB 설치 버전 확인:
    C:\Program Files\MongoDB\Server\ 폴더에서 버전 확인

- cmd에서 실행:
    cd C:\Program Files\MongoDB\Server\8.2\bin
    mongod --dbpath C:\Sohobi\mdb

※ cmd 닫으면 서비스 종료됨!
※ 포트번호 : 27017 (기본값)

-----------------------------------------------------
4. mongosh 설치 [서버에서]
-----------------------------------------------------
- https://www.mongodb.com/try/download/shell
- Windows 64-bit 선택 → Download → 압축 해제
- mongosh.exe 파일을 아래 경로로 복사:
    C:\Program Files\MongoDB\Server\8.2\bin

-----------------------------------------------------
5. mongosh 접속 [서버에서]
-----------------------------------------------------
    cd C:\Program Files\MongoDB\Server\8.2\bin
    mongosh

- 접속 확인:
    Connecting to: mongodb://127.0.0.1:27017/...
    Using MongoDB: 8.2.6
    Using Mongosh: 2.8.1

-----------------------------------------------------
6. DB 생성 & 접속
-----------------------------------------------------
    use DB이름
    use sohobi
    → switched to db sohobi

-----------------------------------------------------
7. 주요 명령어 (Oracle SQL과 비교)
-----------------------------------------------------
7-1. CREATE TABLE → 없음 (자동 생성)

7-2. INSERT
    db.컬렉션명.save({속성명1:값1, 속성명2:값2});
    예) db.product.save({name:'커피', price:1000});

7-3. SELECT
    db.컬렉션명.find();                          ← 전체 조회
    db.컬렉션명.find({속성명:값});                ← 조건 조회
    db.컬렉션명.find({속성명:{연산자:값}});        ← 연산자 조회
    연산자: $lt(미만), $lte(이하), $gt(초과), $gte(이상)

7-4. UPDATE
    db.컬렉션명.update(조건, {$set:{속성명:값}});
    예) db.product.update({name:'커피'}, {$set:{price:500}});

7-5. DELETE
    db.컬렉션명.remove(조건);
    예) db.product.remove({price:{$gt:2000}});
