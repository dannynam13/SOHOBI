===================================================
Oracle 18c XE 설치 & 환경설정 정리
===================================================

-----------------------------------------------------
1. Oracle 18c XE 서버 설치 [서버에서]
-----------------------------------------------------
- OracleXE184_Win64.zip 압축 해제 후 setup.exe 실행
- 설치 중 SYS/SYSTEM 관리자 비밀번호 설정 (메모해둘 것!)
- 비밀번호는 나중에 변경 가능:
    alter user 아이디 identified by 새비번;

-----------------------------------------------------
2. DB 파일 저장 폴더 확보 [서버에서]
-----------------------------------------------------
- Oracle XE 설치 폴더와 별도로 데이터 파일 폴더를 따로 만들 것
- 예) C:\Sohobi\odb

-----------------------------------------------------
3. 통신 정보 확인 [서버에서]
-----------------------------------------------------
- IP 주소 : 10.1.92.119 (시작 - cmd - ipconfig - IPv4 주소)
- 포트 번호 : 1521
- DB 이름 : xe
- 방화벽 OFF

-----------------------------------------------------
4. 서버 통신 여부 확인 [PC에서]
-----------------------------------------------------
    시작 - cmd - ping 서버IP주소

-----------------------------------------------------
5. 관리자로 접속 [서버에서]
-----------------------------------------------------
    시작 - cmd
    sqlplus / as sysdba

-----------------------------------------------------
6. 18c 계정 생성 시 이전 문법으로 전환
-----------------------------------------------------
    alter session set "_ORACLE_SCRIPT"=true;

※ Oracle 11g → 18c 문법 변경으로 c## 접두사 필요
   위 명령어로 이전 문법으로 되돌려서 사용

-----------------------------------------------------
7. 계정 생성 [서버 or 관리자 계정에서]
-----------------------------------------------------
7-1. 이전 문법 전환
    alter session set "_ORACLE_SCRIPT"=true;

7-2. 계정 생성
    create user 아이디 identified by 비번;
    create user shobi identified by "8680";
    ※ 숫자로만 된 비밀번호는 쌍따옴표 필수!

7-3. 권한 부여
    grant connect, resource to 아이디;
    grant connect, resource to shobi;

7-4. tablespace 생성
    create tablespace shobiTS
    datafile 'C:\Sohobi\odb\shobiTS.dbf'
    size 500m;
    ※ 폴더(C:\Sohobi\odb)가 미리 존재해야 함!

7-5. 계정에 tablespace 연결 및 용량 제한 해제
    alter user shobi
    default tablespace shobiTS
    quota unlimited on shobiTS;

-----------------------------------------------------
8. 계정 목록 확인
-----------------------------------------------------
    select username from dba_users;

-----------------------------------------------------
9. instantclient 설치 [PC에서]
-----------------------------------------------------
https://www.oracle.com/database/technologies/instant-client/winx64-64-downloads.html
- Basic Package
- SQL*Plus Package
- JDBC Supplement Package
  위 3개 받아서 한 폴더에 압축 해제
  예) instantclient_23_9

-----------------------------------------------------
10. 원격 접속 [PC에서]
-----------------------------------------------------
    instantclient 폴더에서 cmd 열고:
    sqlplus shobi/"8680"@10.1.92.119:1521/xe

-----------------------------------------------------
11. DBeaver 설치 & 연결 [PC에서]
-----------------------------------------------------
- 다운로드: https://dbeaver.io/download/ (Community Edition)
- 설치 후 New Database Connection → Oracle 선택
- 연결 정보:
    Host     : 10.1.92.119
    Port     : 1521
    Database : xe
    Username : shobi
    Password : 8680
- Test Connection → Connected 확인 후 Finish
- SQL 편집기 단축키: F3 또는 Ctrl+]
- 쿼리 실행: Ctrl+Enter (한 줄) / Alt+X (전체)

-----------------------------------------------------
12. CMD 우클릭 붙여넣기 설정
-----------------------------------------------------
- CMD 타이틀바 우클릭 → 속성 → 옵션 탭
- "빠른 편집 모드" 체크 → 확인
- 이후 드래그=복사 / 우클릭=붙여넣기

-----------------------------------------------------
13. 비밀번호 변경
-----------------------------------------------------
    alter user 아이디 identified by 새비번;
