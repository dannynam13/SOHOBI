1) Git 사용자 설정 (처음 한 번만)
ⓐ 사용자 등록
git config --global user.name "<gitHub_ID명>"
git config --global user.email "<gitHub_이메일명>"
ex) 
git config --global user.name "TerryBlackhoodWoo"
git config --global user.email "leftdeadman@gmail.com"

ⓑ 설정 확인
git config --global user.name
git config --global user.email

2) 작업 후 업로드 과정 (매번 반복)

ⓐ 최신 코드 먼저 받기 (중요 ⭐)
git checkout WOO
git pull origin main

ⓑ (merge 메시지 화면 나오면)
